# MessagePipeTest

メッセージングパイプラインライブラリである[MessagePipe](https://github.com/Cysharp/MessagePipe)のUnity学習用プロジェクト

## Getting Started

cloneした後Unity2021.1.3f1で起動してください。

以下のパッケージが自動でインポートされます。

* MessagePipe
* UniRx
* UniTask
* VContainer

MessagePipeTestXXXフォルダにTestSceneXXXが含まれています。
